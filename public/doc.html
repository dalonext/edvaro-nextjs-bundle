# Edavro – Online Learning Platform Next.js Documentation Edavro is a complete online education platform built with **Next.js**, including both a **frontend mobile-first PWA learning app** and an **admin dashboard panel**. The project is designed for eLearning platforms, online course marketplaces, education SaaS products, training portals, and digital learning businesses. It provides a complete user experience for students on the frontend and a powerful management system for administrators on the dashboard side. Edavro was originally designed as a Figma UI kit and has now been converted into a production-ready **Next.js interface**, making it easier for developers to customize, extend, and integrate with backend APIs. --- # Project Overview Edavro consists of two main parts: ## 1. Frontend PWA Mobile App The frontend is a **mobile-first Progressive Web App** designed for learners. It provides a smooth course discovery, enrollment, payment, learning progress, certification, messaging, profile, and support experience. The app is built with a modern dark-mode visual style and is optimized for mobile devices. It can be used as a web-based mobile learning platform and can also be installed as a PWA on supported devices. ## 2. Admin Dashboard Panel The admin dashboard is a desktop-focused management panel for platform owners, admins, and education teams. It helps manage students, courses, revenue, transactions, certificates, chat, and platform performance from one centralized interface. The dashboard uses a clean sidebar navigation system, structured tables, performance cards, data visualization sections, course management screens, and admin controls. --- # Technology Stack The Edavro Next.js project uses a modern frontend architecture. ## Core Technologies * Next.js * React * JavaScript or TypeScript, depending on project setup * HTML5 * CSS3 / SCSS / Tailwind CSS, depending on implementation * Responsive layout system * Component-based UI structure * Progressive Web App support for frontend mobile app ## Font * Roboto ## Design Source * Edavro Education Online Course Mobile App UI Kit * Edavro Online Learning Management Dashboard UI Kit --- # Frontend PWA Mobile App The frontend app is designed for students and learners. Installation Install dependencies: ```bash npm install ``` Run the development server: ```bash npm run dev ``` Open the project in your browser: ```txt http://localhost:3000 ``` Build for production: ```bash npm run build ``` Start the production server: ```bash npm start ``` --- #
