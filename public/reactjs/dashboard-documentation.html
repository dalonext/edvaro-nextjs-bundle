Edavro — LMS Dashboard (React)

A fully production-grade Learning Management System admin dashboard built with React 19, TypeScript 6, Vite, Tailwind CSS 3, and Redux Toolkit. Complete state management with 8 Redux slices, 50+ UI components, 10 custom hooks, and comprehensive documentation.

Installation & Setup

Install dependencies Run npm install to get all packages 
npm install
Start the development server 
npm run dev

Visit http://localhost:5173 in your browser.

Build for production 
npm run build
npm run preview
ℹ️ This project uses mock JSON data in src/data/*.json. No backend required.

Key Features

🔐 Authentication System
Complete auth flow with login, signup, OTP verification, and password reset.
📊 Dashboard Analytics
KPI cards, charts, and real-time analytics with Recharts integration.
📚 Course Management
Full course CRUD operations, video player, modules, and ratings.
👥 Student Management
Enrollment tracking, progress monitoring, and student profiles.
💰 Transaction Management
Payment tracking, invoicing, and financial reporting.
📜 Certificate Generation
PDF certificate download with jsPDF and html2canvas.
💬 Chat System
Real-time messaging interface with conversation history.
🎨 Design System
Complete design tokens, 3-way theme, and Tailwind CSS config.
